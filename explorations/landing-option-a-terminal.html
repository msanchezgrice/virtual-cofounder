// SCANNING SYSTEM

6 comprehensive scan types

From SSL certs to Core Web Vitals. Deep analysis of every project, every day.

01

Domain Health

HTTP/HTTPS reachability, SSL certificate validation, DNS resolution, redirect chains, response time monitoring.

02

SEO Analysis

Title, meta description, OG tags, H1 quality, canonical URLs, robots.txt, sitemap.xml, Schema.org markup.

03

Analytics Detection

PostHog, Google Analytics, Plausible, Fathom snippet detection. Tag Manager presence verification.

04

Vercel Deployment

Latest deployment status (READY/ERROR/BUILDING), build duration, errors, missing environment variables.

05

Playwright Deep Scans

Core Web Vitals (FCP, LCP, CLS, TBT), Lighthouse scores, screenshots, console errors, network waterfall.

06

Security Scanning

Exposed API keys (Stripe, OpenAI, etc.), npm audit, outdated packages, missing security headers (CSP, HSTS).
