Six comprehensive scan types

From SSL certificates to Core Web Vitals. Deep analysis of every project, executed daily at 9am or on-demand.

01

Domain Health

HTTP/HTTPS reachability, SSL validation, DNS resolution, redirect chains, response time monitoring.

02

SEO Analysis

Title, meta description, OG tags, H1 quality, canonical URLs, robots.txt, sitemap.xml, Schema.org markup.

03

Analytics Detection

PostHog, Google Analytics, Plausible, Fathom snippet detection. Tag Manager presence verified.

04

Vercel Deployment

Deployment status (READY/ERROR), build duration, errors, missing environment variables, preview URLs.

05

Playwright Deep Scans

Core Web Vitals (FCP, LCP, CLS), Lighthouse scores, screenshots, console errors, network waterfall analysis.

06

Security Scanning

Exposed API keys (Stripe, OpenAI, etc.), npm audit, outdated packages, missing security headers (CSP, HSTS).
